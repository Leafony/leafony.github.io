STM32 MCU の Arduino IDE 設定

Arduino IDEにSTM32 Coresをインストール

以下の手順で STM32 MCU 開発環境の設定を行いましょう。

用意するもの

  • STM32 MCUリーフ
  • USBリーフ
  • PC (Windows、Mac OS X 、または Linux)

Arduino IDEはインストールしてある前提です。

マイコンボードの設定

STM32 MCUリーフをArduino IDEで使用するには、STM32 Coresのインストールが必要です。

Arduino IDEのファイル → 環境設定を開き、追加のボードマネージャのURL:に、下記URLを入力してください。

https://github.com/stm32duino/BoardManagerFiles/raw/master/STM32/package_stm_index.json

ツール→ボード→ボードマネージャを選択し、STM32 Coresを検索し、インストールのボタンを押しましょう。

STM32 Coresの詳細は、こちらを参照してください。

STM32CubeProgrammer software V2.4.0をインストールすれば、STM32 MCUが使えるようになります。

V2.5.0は動かないので注意してください。

Get Softwareボタンをクリックします。

Select versionsをクリックして、2.4.0を選択します。

ACCEPTをクリックします。

名、姓、E-mail addressを入力し、
□I have read and understood the Sales Terms & Conditions, Terms of Use and Privacy Policy
をチェックして、Downloadをクリックします。

下記が表示されます。

登録したアドレスにメールが届くのでDownload nowをクリックしてダウンロードします。

マイコンボードへの書き込み

ツール → ボード→ STM32 Boards (Selected from submenu)→ Nucleo-64 をクリックします。

ツール → Board part number: *****→ Nucleo L452RE をクリックします。

ツール → Upload method: ***** → STM32CubeProgrammer(Serial) をクリックします。

これでSTM32 MCUリーフが使えるようになります。

マイコンボードに書き込む

STM32 MCUリーフの書き込みスイッチをOnにします。すると赤LEDが点灯します。 Arduino IDEのマイコンボードに書き込むボタンをクリックします。 正常に書き込まれた場合は Start operation achieved successfullyと表示されます。

書き込みが完了したら、 STM32 MCUリーフの書き込みスイッチをOffにしてください。すると赤LEDが消灯します。 STM32 MCUリーフのリセットスイッチを押してください。

書き込みスイッチをOnにしないで書き込みを行った場合は ボードへの書き込みが完了しましたと表示されますが、下部に Timeout error occuredと表示されます。

ピンの指定

STM32 MCUのピン配置は、Nucleo L452REと異なっていますので、STM32L452REのポート名で使用するピンを指定してください。

スケッチの違い

Timer

AVR MCUで使用する”MsTimer2”はSTM32 MCUでは使用できません。 HardwareTimer等を使用してください。 使用例を以下に記載します。 詳細はこちら を参照してください。

初期設定

HardwareTimer *timer2 = new HardwareTimer (TIM2);

  timer2->setOverflow(LOOP_INTERVAL, MICROSEC_FORMAT);       // 125ms
  timer2->attachInterrupt(intTimer);
  timer2->resume();
