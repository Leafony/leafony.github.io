AC08 LTE-M Mary


特徴

（仕掛中）LTE-M Maryリーフは、xxが共同開発したLTE-M (LTE Cat-M1)通信機能に対応したリーフです。

  • 約23×30mmの小さな基板上に、データ通信用のアンテナを内蔵し（技適取得済）、外部アンテナは不要です。
  • SIMカードは、 KDDIのLTE-M通信網に対応したソラコムのLTE-Mサービス「SORACOM Air for セルラー(plan-KM1)」が利用出来ます。
  • 省電力性能は、通信モジュールが具備するスリープ等の機能だけでなく、ソフトウェア設定で通信モジュールへの給電自体を遮断する機能を搭載しており、長期間使用しない時は電池の消費を限りなくゼロに近づけることが可能にしました。
  • 電池は、単3電池やリチウムイオン電池等で、データ送信の頻度や回数等のユースケースに応じて選択することが出来ます。

AT Commands

ATコマンドは、3GPP（携帯通信の国際標準団体）によって標準化され、モデムの動作や接続先を設定出来ます。 モデムによって仕様が異なるため、メーカーが提供するATコマンドのマニュアルを確認することが重要です。
以下に、SORACOMを例に、接続する手順を記します。

用意するもの

  • LBAD0XX1SC-DM-EVK-B
  • パソコン (Windows、または macOS)
  • 標準SIMカード(例　SORACOM plan-D)
  • APNの設定
    • APN:soracom.io
    • ユーザー名: sora
    • パスワード: sora
    • PDP Type: IP
    • 認証タイプ: CHAP または PAP
  • MCC(Mobile Country Code)とMNC(Mobile Network Code)
Type plan-D plan-K2
MCC 440 440
MNC 10 51

AT Commands Basics

ATコマンドは、通常 ATにで始まり、その後に <command>、場合によっては、 <value>が続きます。AT コマンドには、 Set、Read、Test、Executeの 4種類に分類されます。

Type Syntax Extample Description
Set AT+<command>=<value> AT+COPS=1,2,<MCCMNC> ネットワーク事業者に手動で接続 　　　　　
Read AT+<command>? AT+COPS? 現在使用中のネットワーク事業者を確認
Test AT+<command>=? AT+COPS=? 利用可能なネットワーク事業者を検索
Execute AT+<x> AT+CSQ 現在の信号強度と接続の品質を数値で確認
