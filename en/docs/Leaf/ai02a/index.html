AI02A SP&PIR

Features

The Leaf is equipped with a piezoelectric speaker and a human-detecting sensor. The piezoelectric speaker is connected to the MCU leaf by PWM and the human-detecting sensor by I2C. The human-detecting sensor has a built-in proximity detection algorithm that can detect proximity with simple settings, and can output an interrupt signal when proximity is detected.
