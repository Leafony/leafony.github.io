Basic Kit スタートガイド

Leafonyで環境センシングを始めよう！

Leafony Basic Kitには、温湿度、照度、加速度センサを搭載した4-Sensorsリーフと、AVR MCUリーフ、Bluetooth LEリーフ、CR2032コイン電池リーフを組み合わせた、2cm立方のIoTデバイスが入っています。

Bluetoothを使ってスマートフォンやパソコンにセンサの情報を送信したり、Leafonyに搭載されたLEDをコントロールすることが簡単にできます。

Basic Kitに含まれるLeafonyには、箱を開けたらすぐに使えるサンプルアプリが書き込まれています。

Note

Web Bluetoothを使ったWebアプリで、Bluetooth接続もプラットフォームを問わずインストール不要で動作します。

用意するもの

  • Leafony Basic Kit
  • Windows1、 Mac2、 Linux3、 Chrome OSのPCまたは
    Android4、 iPhone5、 iPad5のスマートフォンまたはタブレット
  • Google Chrome (バージョン70.0.3526.0以降)

Androidの場合

  1. Basic Kitに電池を挿入し、電源スイッチをONにする


  1. Google Chromeを開く
  2. サンプルWebアプリのページを開く
  3. WebアプリのConnectボタンを押して、Leaf_A_002」を選択し、ペア設定ボタンを押す
  4. 接続完了
  5. サンプルアプリのソースコードはこちら

次のステップ

開発環境設定を読んで、LeafonyにLED点滅スケッチを書き込んでみましょう。

iPhoneまたはiPadの場合

  1. Basic Kitに電池を挿入し、電源スイッチをONにする


  1. App StoreからWebBLE (有償)をダウンロード5
  2. WebBLEを立ち上げ、下記のURLにアクセス
    https://docs.leafony.com/WebBluetooth_for_Leafony_app/
  3. WebアプリのConnectボタンを押して、Leaf_A_002」を選択し、ペア設定ボタンを押す


注) 画面上に「Download CSV 」のボタンはありますが、iPhoneでは使用できません。

次のステップ

開発環境設定を読んで、LeafonyにLED点滅スケッチを書き込んでみましょう。
